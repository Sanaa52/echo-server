Файлы
server.py - TCP-сервер, который принимает соединения и возвращает полученные сообщения обратно клиенту (эхо-сервер)

client.py - TCP-клиент, который подключается к серверу и отправляет сообщения

Требования
Python 3.6 или выше

Сетевое подключение между клиентом и сервером (или локальный хост для тестирования)

Установка и запуск
1. Запуск сервера
bash
python server.py
Сервер запустится на всех доступных сетевых интерфейсах (0.0.0.0) и порту 5000. После запуска сервер будет ждать подключений от клиентов.

2. Запуск клиента
Перед запуском клиента убедитесь, что в файле client.py указан правильный IP-адрес сервера:

python
HOST = "10.90.14.76"  # Замените на IP-адрес вашего сервера
Запустите клиент:

bash
python client.py
Использование
Запустите сервер на одной машине

Запустите клиент на другой машине (или на той же, если тестируете локально)

В клиенте вводите сообщения, которые будут отправлены на сервер

Сервер получит сообщение и отправит его обратно клиенту

Для завершения соединения введите exit

Особенности
Сервер работает как эхо-сервер: возвращает клиенту те же данные, которые получил

Поддерживается кириллица и другие Unicode-символы (используется кодировка UTF-8)

Для завершения соединения клиент отправляет сообщение "exit"

Сервер может обрабатывать только одно подключение одновременно

Настройка
Изменение порта
Чтобы изменить порт, отредактируйте переменную PORT в обоих файлах:

python
PORT = 5000  # Измените на нужный порт
Изменение IP-адреса сервера
В файле client.py измените переменную HOST:

python
HOST = "127.0.0.1"  # Для локального тестирования
# или
HOST = "192.168.1.100"  # IP-адрес вашего сервера в сети
Возможные проблемы и решения
"Connection refused" - убедитесь, что сервер запущен и порт не заблокирован фаерволом

"Address already in use" - порт занят другим приложением, измените порт или завершите другое приложение

Клиент не может подключиться - проверьте IP-адрес сервера и убедитесь, что клиент и сервер находятся в одной сети

Пример использования
text
# На сервере
$ python server.py
Connected by ('192.168.1.100', 54321)

# На клиенте
$ python client.py
Введите сообщение (exit для выхода): Привет, сервер!
Ответ сервера: Привет, сервер!
Введите сообщение (exit для выхода): exit
